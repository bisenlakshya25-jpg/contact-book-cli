# 📞 Contact Book CLI

A command-line Contact Book application built using Python. It allows users to manage contacts efficiently with persistent JSON storage, search functionality, and complete CRUD operations.

---

## ✨ Features

- ➕ Add new contacts
- 📋 View all contacts
- 🔍 Search contacts using partial name or phone number
- ✏️ Update contact information
- 🗑️ Delete contacts
- 💾 Persistent data storage using JSON
- 📱 Phone number validation
- 📧 Email validation
- 🔤 Automatic alphabetical sorting
- 🔎 Handles multiple matching contacts gracefully
- 🧩 Modular and reusable code structure

---

## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- OS Module

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/contact-book-cli.git
```

2. Open the project folder

```bash
cd contact-book-cli
```

3. Run the program

```bash
python main.py
```
